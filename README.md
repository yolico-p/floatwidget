  <h2 style="color: #00d4ff; text-align: center; letter-spacing: 0.05rem;">项目介绍</h2>

    <p style="color: #ccd6f6; line-height: 1.7; font-size: 1.05rem; margin: 0;">
      WinFloat Widget 是一款专为 Windows 平台打造的轻量级桌面悬浮窗工具，基于 Qt5 框架原生开发。
      支持自定义文字、图片的桌面悬浮显示，提供窗口拖拽、位置锁定、开机自启等核心能力，同时内置智能分辨率适配、多屏环境兼容机制，默认限制图片最大尺寸为多屏中最小屏幕分辨率的80%，从根源避免“赛博灯泡”式的视觉干扰，为用户提供简洁、稳定、高度自定义的桌面悬浮解决方案。
    </p>
